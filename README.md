# BSC 钱包资产分析 + 三重摆混沌模拟

## 项目文件

### 1. `bsc_balance.html` — BSC 钱包全量资产扫描
**提示词：**
> "你给我写一个脚本：我给你一个加密钱包的地址，你把这个钱包中 BSC 链上的余额读取出来"
> "我希望能把这条链上的所有主流币的价值都给出来，按 USDT 余额表示"
> "没有给出该地址持有的全部链上资产（BNB 主币 + BEP20 代币），统计资产种类与估值"

**功能：** 输入 BSC 钱包地址，自动扫描该地址持有的所有代币余额并显示 USDT 估值。

**技术架构：**
- 5 层探测策略：1inch API → 链上 Transfer 事件扫描 → PancakeSwap 代币列表 → 预设主流代币 → 用户自定义代币
- 价格获取：外部 API（Binance/MEXC/备用） → 链上 PancakeSwap Router 查询
- JSON-RPC Batch 批量查询优化
- 支持手动添加任意代币合约地址

### 2. `bsc_balance.py` — BSC 钱包余额查询（Python CLI 版）
**提示词：** 同上（初始版本）

**功能：** 命令行查询 BSC 钱包的 BNB 余额。纯 Python 标准库实现，无需第三方依赖。

### 3. `triple_pendulum.html` — 三重摆混沌模拟
**提示词：**
> "使用原始 HTML5 画布模拟真实的物理现象。提示 1：'一个三重摆摆动进入混沌，用其尖端绘制发光的轨迹'"

**功能：** 物理模拟三重摆的混沌运动，用摆尖绘制发光轨迹。

**技术实现：**
- RK4 数值积分求解拉格朗日动力学方程
- 3x3 质量矩阵 + 高斯消元求解角加速度
- 径向渐变辉光 + shadowBlur 发光效果
- 渐变彩色轨迹（红橙到黄到白）
- 点击画布随机重置初始状态

---

## 使用方式

### BSC 钱包扫描
直接在浏览器打开 `bsc_balance.html`，输入钱包地址即可。

### 三重摆模拟
直接在浏览器打开 `triple_pendulum.html`，点击画布切换混沌模式。
